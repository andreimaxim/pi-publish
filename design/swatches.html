Pierre keeps #009fff, #00cab1, #ff2e3f, #ffca00 identical across dark and light themes. This works because Pierre uses them for tiny syntax tokens and line indicators. Our template uses --orange and --teal for inline reading ‚Äî bold phrases and code spans inside paragraphs ‚Äî so those two must be darkened to remain legible.
